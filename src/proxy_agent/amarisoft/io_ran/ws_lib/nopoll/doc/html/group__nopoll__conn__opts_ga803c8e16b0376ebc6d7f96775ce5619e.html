void nopoll_conn_opts_free ( noPollConnOpts *  opts)

Allows to release a connection object reported by nopoll_conn_opts_new.

IMPORTANT NOTE: do not use this function over a noPollConnOpts if it is not flagged with nopoll_conn_opts_set_reuse (opts, nopoll_true).

Default behaviour provided by the API implies that every connection options object created by nopoll_conn_opts_new is automatically released by the API consuming that object.

Referenced by nopoll_conn_opts_unref().
