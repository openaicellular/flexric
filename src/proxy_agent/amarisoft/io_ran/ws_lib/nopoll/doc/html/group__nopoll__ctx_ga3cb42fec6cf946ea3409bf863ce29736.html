◆ nopoll_ctx_ref()

nopoll_bool nopoll_ctx_ref ( noPollCtx *  ctx)

Allows to acquire a reference to the provided context.

This reference is released by calling to nopoll_ctx_unref.

Parameters
ctxThe context to acquire a reference.
Returns
The function returns nopoll_true in the case the reference was acquired, otherwise nopoll_false is returned.

References nopoll_false, nopoll_mutex_lock(), nopoll_mutex_unlock(), nopoll_return_val_if_fail, and nopoll_true.
