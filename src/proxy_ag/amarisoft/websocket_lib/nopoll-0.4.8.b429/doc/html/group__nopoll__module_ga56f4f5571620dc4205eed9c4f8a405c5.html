nopoll_bool nopoll_ncmp ( const char *  string1,
const char *  string2,
int  bytes 
)

Allows to check if provided strings are equal for the first bytes.

Parameters
string1The first string to check. The string must be ended by 0.
string2The second string to check. The string must be ended by 0.
bytesNumber of bytes to check. The value must be > 0.
Returns
nopoll_true if both strings are equal, otherwise nopoll_false is returned.
