nopoll_bool nopoll_conn_is_tls_on ( noPollConn *  conn)

Allows to check if the provided connection is working under a TLS session.

Parameters
connThe connection where the TLS is being queired to be enabled.
Returns
nopoll_true in the case TLS is enabled, otherwise* nopoll_false is returned. Note the function also returns* nopoll_false when the reference received is NULL.
